
Compiler
========

A script `compile.sh` for compiling is provided in both problems. 

Just run, 

```
$ sh compile.sh 
```

Run
===

The generated executable is named `dict`. 

```
$ ./dict 
``` 

Then cmds such as insert, find, delete, print can be performed through command line. 

Limitation
=========

The implementation of balanced tree, the AVL tree, is incomplete, which lacking remove method. Altough several trials have been made, still fail to make a working implementation of deletion method.
